Data & Excel

Import data dari file Excel monitoring SPPG, ekspor ulang, atau hapus seluruh data.

Import dari Excel

Format kompatibel SPPG

Format yang didukung: .xlsx, .xls, .csv. Header yang dideteksi otomatis: Tanggal, Hari, Menu Nasi/Protein Hewani/Sayur/Protein Nabati/Buah, Jumlah Porsi, Pemakaian & Sampah (kg) tiap item.

Data akan ditambahkan ke catatan saat ini (tidak menimpa).

Export & Reset

Backup atau bersihkan data
Hasil export berisi semua kolom Excel asli + kolom hasil hitung.
Tindakan ini tidak dapat dibatalkan.

Panduan & Parameter Analisis

Aturan perhitungan sesuai sheet "Panduan" pada Excel sumber.

Parameter Penilaian

Ambang minimal serapan agar status = Terserap80%
Ambang maksimal sampah terhadap pemakaian20%
Satuan inputkilogram (kg)
Keterangan statusTerserap jika % Serapan ≥ ambang atau % Sampah ≤ (1 − ambang)

Cara Pakai

  1. Buka menu Input Harian, klik Tambah Data.
  2. Isi tanggal, nama menu, jumlah porsi, total pemakaian (kg), dan jumlah sampah (kg) per item.
  3. Semua kolom % Sampah, % Serapan, dan Status terhitung otomatis untuk tiap item dan total.
  4. Buka Dashboard untuk rekap, ambang batas, dan status akhir.
  5. Buka Grafik Item untuk membandingkan pemakaian vs sampah tiap item.
  6. Gunakan Import Excel untuk memuat data dari file SPPG.
