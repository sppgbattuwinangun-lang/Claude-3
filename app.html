Panduan Penggunaan

              
Cara mengisi data, menafsirkan dashboard, dan membuat laporan PDF.

            

          


          

            
Parameter Penilaian

            

              
    
                
  • Ambang minimal serapan agar status Terserap: 80%
    • 
                
  • Ambang maksimal sampah terhadap pemakaian: 20%
    • 
                
  • Satuan input berat: kilogram (kg)
    • 
                
  • Status Terserap jika % Serapan ≥ ambang minimal, sebaliknya Perlu Evaluasi
    • 
              

            

          


          

            
Cara Pakai

            

              
    
                
  1. Buka halaman Input Harian, klik Tambah Data.
    2. 
                
  2. Isi tanggal, jumlah porsi, nama menu, lalu pemakaian (kg) dan sampah (kg) per item.
    3. 
                
  3. Hari otomatis terisi dari tanggal. Persentase sampah, persentase serapan, dan status dihitung otomatis.
    4. 
                
  4. Lihat ringkasan harian + tren grafis pada halaman Dashboard.
    5. 
                
  5. Untuk laporan: buka Laporan PDF, pilih rentang waktu, tinjau hasilnya, lalu klik Unduh PDF.
    6. 
                
  6. Ganti ambang serapan di Pengaturan bila perlu.
    7. 
              

            

          


          

            
Catatan Keamanan

            

              
Aplikasi ini menyimpan data di localStorage browser Anda. Data tidak dikirim ke server. Untuk bekerja lintas perangkat/admin, gunakan komputer yang sama atau pertimbangkan integrasi backend.

              
Versi aplikasi: —

              
              
Hapus semua data lokal (akun, data harian, pengaturan). Aplikasi kembali ke kondisi default.
